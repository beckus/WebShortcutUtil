WebShortcutUtil

Web Shortcut Utility Suite

Overview

This is a suite of libraries and tools for working with web shortcuts. Web shortcuts are the files that are created by various operating systems when you save a link to a website.

I have found myself using various operating systems over the years, and I also have been increasingly using cloud file services. I sometimes work in one operating system, create a shortcut, put the shortcut on a cloud service, then try to open the shortcut in another operating system. This generally does not work. There are workarounds such as opening the shortcuts with a text editor and copying the URL, or using link services such as Delicious.

I decided to try and create a set of applications and libraries that would allow you to open any shortcut regardless of which operating system you are using. A major goal was to gain experience using various programming languages, installers, etc., and this project seemed ideal, since the libraries/applications would be fairly small. I originally intended to create installable applications for Windows, Linux, and maybe Apple, but never did. There is a lot of room for improvement, but I no longer will have the time.

The functionality is as follows:

  • Be able to retrieve the name and URL of Windows, Linux, and Apple shortcuts. Specifically these are files with the following extensions: .url, .website, .desktop, .webloc.
  • Be able to create shortcut files given a name and URL.
  • More details can be found in the library documentation.

Below you will find a list of the applications/libraries.

Launcher for PCs

This is a command line Java application that allows you to launch shortcuts on your PC. Unfortunately, due to differences in operating systems, it is difficult to open a browser reliably in Java. Therefore, this application will not work on all operating systems or distributions.

This Java application could be packaged into installers for the various operating systems, but I have no plans to do so. Instead, I have a wiki page with manual installation instructions, which are unfortunately cumbersome and incomplete.

  • Installation wiki page: https://github.com/abcodeworks/LaunchWebShortcut-Java/wiki/Installing/
  • Source code: https://github.com/abcodeworks/LaunchWebShortcut-Java/

Android App

This is an Android app that allows you to launch and create shortcuts.

  • Android Marketplace Page:
  • Source code: https://github.com/abcodeworks/WebShortcut-Android/

Java Library

This library has been tested on Android and appears to be completely compatible. The functionality and implementation is largely equivalent to the Java library.

  • Source code: https://github.com/abcodeworks/WebShortcutUtil-Java/
  • Maven Repository:
  • Javadocs: http://abcodeworks.github.io/WebShortcutUtil-Java/javadoc/latest/

Perl Library

This is a PERL library. The functionality and implementation is largely equivalent to the Java library.

  • Source code: https://github.com/beckus/WebShortcutUtil-Perl/
  • CPAN module: http://search.cpan.org/~beckus/WebShortcutUtil/

Samples

During development I collected a set of shortcuts from various operating systems (both to study and to use for unit tests). I made these available in their own repository.
