WebShortcutUtil

Web Shortcut Utility Suite

Overview

This is a suite of libraries and tools for working with web shortcuts. Web shortcuts are the files that are created by various operating systems when you save a link to a website.

I have found myself using various operating systems over the years, and I have also increasingly used cloud file services. I sometimes work in one operating system, create a shortcut, put the shortcut on a cloud service, then try to open the shortcut in another operating system. This generally does not work. There are workarounds such as opening the shortcuts with a text editor and copying the URL, or using link services such as Delicious.

I decided to try and create a set of applications and libraries that would allow you to open any shortcut regardless of which operating system you are using. I don't find this subject particularly exciting, so I was initially hesitant to start a new project. However, I wanted to gain experience using various programming languages, installers, etc., and This project seemed ideal due to its small size.

Functionality

  • Supports shortcut files for Windows, Linux, and Apple. Specifically, these are files with the following extensions: .url, .website, .desktop, .webloc.
  • Retrieves the name and URL from shortcut files.
  • Creates shortcut files given a name and URL.
  • More details can be found in the library documentation.

Applications and Libraries

Java Launcher for PCs

A command line Java application that allows you to launch shortcuts on your PC. Please see the Installation wiki page for more details.

I also hope to create an application for converting shortcuts between the various types (but I have no plans to do so in the near future).

Known issues

  • The launcher does not work on some operating systems, for example Solaris.
  • The launcher only works on Linux distributions that have the "xdg-open" command installed.
  • The launcher does not appear to work on Windows with files with unicode characters (it gives a "file not found" error).

Java Library

A Java library intended for use in other Java applications. It provides classes/methods for reading and writing shortcuts. This library has been tested on Android and appears to be completely compatible. The functionality and implementation is largely equivalent to the Perl library.

The library is published to the Maven repository. Use the following dependency:

<dependency>
    <groupId>com.abcodeworks</groupId>
    <artifactId>webshortcututil</artifactId>
    <version>1.0.0</version>
</dependency>
